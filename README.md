# OffTick

OffTick 是一个 macOS 菜单栏小工具，用一个小小的悬浮窗展示日期、时间、天气、空气质量、下班倒计时和实时收入。

## 当前雏形

- 菜单栏按钮：点击显示或隐藏悬浮窗
- 悬浮窗：置顶显示核心信息，可拖动
- 日期与时间：每秒刷新
- 下班倒计时：默认按电脑开机时间 + 8 小时计算
- 收入跳动：默认按月薪 10000、当月 22 个工作日、每天 8 小时计算
- 天气 / 空气质量：已预留展示位，后续接 API

## 运行

```bash
swift build
.build/debug/OffTick
```

## 打包运行

```bash
./Scripts/build-app.sh
open .build/OffTick.app
```

菜单栏应用建议通过 `.app` 启动。脚本会生成稳定的 Bundle ID（`dev.local.OffTick`）并在写入 `Info.plist` 后对整个 app 重新签名，避免系统设置里出现多个同名 OffTick 或开关不对应当前运行版本。
